
Holidae Website – Web Development Project

Description
This project is a responsive travel website developed using HTML, CSS, and JavaScript.
The website allows users to browse travel packages, view package details, compare deals, save favourites, and make bookings.

The project demonstrates front-end web development skills, including layout design, responsive styling, navigation, and basic JavaScript functionality.

Technologies Used
HTML5
CSS3 (Responsive Design)
JavaScript (Vanilla JS)

## How to Run the Project
1. Download or unzip the project folder.
2. Open the folder.
3. Double-click on `index.html`.
4. The website will open in your web browser.
